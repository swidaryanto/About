# About — Curated Motion Playground

Interactive motion playground. Test draggable toast layouts, spinner states, and micro-interactions with clean reload defaults.

This repository contains a lightweight interactive playground used to explore and refine UI motion details for the personal site.

## What’s Inside

- A minimal motion-focused interface with draggable UI blocks.
- A toast panel with transition presets (`slide`, `fade`, `scale`).
- Loading indicator experiments:
  - spinner card
  - braille-style CLI spinner
  - trail/ghost motion variations
- Micro-interaction polish for timing, spacing, and visual hierarchy.
